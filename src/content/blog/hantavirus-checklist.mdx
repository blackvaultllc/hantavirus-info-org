# Hantavirus Prevention Checklist: Your Complete Safety Guide

Prevention is the most effective strategy against Hantavirus. This comprehensive checklist covers all aspects of prevention to help you protect yourself and your family. Print this checklist and use it regularly to maintain a safe environment.

## Home Safety Checklist

### Exterior Inspection

**Monthly Tasks:**
- [ ] Check foundation for cracks or holes
- [ ] Inspect vents and screens for damage
- [ ] Look for rodent burrows around foundation
- [ ] Check for rodent droppings near entry points
- [ ] Inspect weather stripping on doors

**Seasonal Tasks:**
- [ ] Fall: Seal any new cracks before winter
- [ ] Spring: Check for winter damage
- [ ] After storms: Inspect for new damage
- [ ] Before winter: Ensure all entry points sealed

### Interior Inspection

**Monthly Tasks:**
- [ ] Check basement and crawl spaces for droppings
- [ ] Inspect attic for rodent signs
- [ ] Check pantry for rodent activity
- [ ] Look for droppings in garage
- [ ] Inspect under sinks for signs

**Weekly Tasks:**
- [ ] Check kitchen for food spills
- [ ] Inspect pet food storage areas
- [ ] Check for any new droppings
- [ ] Ensure food is stored properly

### Food Storage

**Daily Habits:**
- [ ] Store all food in sealed containers
- [ ] Clean up food spills immediately
- [ ] Don't leave pet food out overnight
- [ ] Keep countertops clean
- [ ] Secure garbage cans

**Weekly Tasks:**
- [ ] Check pantry for opened packages
- [ ] Inspect pet food storage
- [ ] Clean out refrigerator
- [ ] Check garbage cans for damage

### Waste Management

**Daily:**
- [ ] Empty kitchen trash regularly
- [ ] Use trash cans with tight lids
- [ ] Don't let trash overflow
- [ ] Clean trash cans if needed

**Weekly:**
- [ ] Clean trash cans with disinfectant
- [ ] Check for rodent activity around cans
- [ ] Ensure lids are secure
- [ ] Replace damaged cans

## Rodent Control Checklist

### Prevention

**Year-Round:**
- [ ] Remove yard debris
- [ ] Keep grass trimmed
- [ ] Store firewood off ground
- [ ] Remove potential nesting sites
- [ ] Keep outdoor areas clean

**Monthly:**
- [ ] Inspect yard for rodent activity
- [ ] Check compost pile
- [ ] Inspect outdoor buildings
- [ ] Check around foundation
- [ ] Look for burrows

### Trapping

**If Rodents Present:**
- [ ] Use snap traps instead of poison
- [ ] Place traps along walls
- [ ] Use appropriate bait (peanut butter, oatmeal)
- [ ] Check traps daily
- [ ] Wear gloves when handling traps

**Safety:**
- [ ] Never use poison where pets can access
- [ ] Dispose of dead rodents properly
- [ ] Clean traps after use
- [ ] Wash hands after handling traps
- [ ] Keep traps away from children

### Professional Help

**When to Call Professional:**
- [ ] Heavy infestation
- [ ] Repeated infestations
- [ ] Hard-to-access areas
- [ ] Large commercial buildings
- [ ] If DIY methods fail

## Cleaning Safety Checklist

### Before Cleaning

**Preparation:**
- [ ] Ventilate area for 30+ minutes
- [ ] Wear N95 mask or better
- [ ] Wear rubber gloves
- [ ] Wear protective eyewear
- [ ] Prepare disinfectant solution

**Supplies Check:**
- [ ] Disinfectant (bleach solution or commercial)
- [ ] Spray bottle
- [ ] Paper towels
- [ ] Plastic bags (heavy-duty)
- [ ] Mop and bucket

### During Cleaning

**Process:**
- [ ] Wet all contaminated surfaces
- [ ] Wait 10 minutes for disinfectant to work
- [ ] Clean with wet towels or mop
- [ ] Avoid sweeping or vacuuming
- [ ] Double-bag all waste

**Safety:**
- [ ] Keep area ventilated
- [ ] Don't touch face while cleaning
- [ ] Change gloves if torn
- [ ] Take breaks if needed
- [ ] Keep children away from area

### After Cleaning

**Cleanup:**
- [ ] Disinfect area again
- [ ] Remove PPE properly
- [ ] Wash hands thoroughly
- [ ] Dispose of waste properly
- [ ] Clean equipment used

**Monitoring:**
- [ ] Monitor for symptoms for 8 weeks
- [ ] Note any health changes
- [ ] Seek care if symptoms develop
- [ ] Inform doctor about exposure
- [ ] Keep record of cleaning date
